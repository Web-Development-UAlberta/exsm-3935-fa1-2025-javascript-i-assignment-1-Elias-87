# University of Alberta Browser Console Template Repository
## Full Stack Web Application Development Program
### This repository is intended for the development of console JavaScript applications.

Run `npm run lint` to attempt to detect errors in your repository.

Use `output()` and `await input()` from `./src/js/scripts.js` to allow the user to interact with the console.
To run the console, launch `./src/index.html`.

Project Title: Replace a's with o's, return modulus

Project description:

User is prompted to input first and last names, is returned full name with all a's replaced with o's.
User is promted to enter large and small numbers, is returned modulus of the large number divided by the small number.
Loop conditions placed on number inputs until conditions are met.
Try block in place to catch error if user puts zero for divisor.

Tech stack:
Javascript console provided by instructor
